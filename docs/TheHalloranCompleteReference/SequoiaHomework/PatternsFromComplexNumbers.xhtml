Problem 1. Use de Moivre's theorem to obtain solutions to the equation z 3 − 1 = 0.

The solutions for z 3 − 1 = 0 are expected to have roots that are complex numbers, of the form a + bi. Complex numbers, such as the solutions for this equation, can be represented in a plot of a coordinate systerm (also called Argand diagram) which represents a complex plane. The real number component magnitude a of the complex number is positioned on the horizontal axis, and the imaginary number component magitude b is positions on the vertical axis, of this plot representation. The value z can be represented in polar coordinate with a vector r with origin at the center (0,0), and the vertex at (a, b), or for any nonzero a magnitude, a units to the left if a < 0, or to the right if a > 0 from the vertical axis, the zero point for real numbers; and for any nonzero b magnitude, b units down if b < 0, or up if b > 0, from the horizontal axis, the zero point for imaginary numbers.

If z is a complex number, or represents a variable whose solutions can involve real numbers (or complex numbers whose value b = 0) as well as complex numbers, then its real and imaginary components can be represented in polar coordinates:

a =r cos  θ b =r sin  θ

Then z can be expressed as: z =a+bi=r  (cos θ +i sin θ )

For the value zn, then this must be true: zn =[r( cos θ+i  sinθ )]n

The formula of de Moivre has proven that this relationship is true: ( cos θ+i  sin θ )n=cos  nθ+i  sin nθ

And so it follows that: z3 ==r3( cos 3θ+i  sin 3θ )

Because r is positive real number, then r = 1. Moreover cos 3θ = 1 and sin 3θ = 0.

The value of a root at the n power, ωn of complex number z for n > 0, will according to de Moivre's theorem be: ωn=[ρ (cos φ +i sin  φ)]n =ρn( cos nφ +i sin  nφ) with a modulus ρ and a phase φ.

As ωn = z, then ρn( cos nφ+i  sin nφ )=r(cos θ+i sinθ )

This equation shows the following corresponding component equalities: ρ n=r   ⇒  ρ= r1/n cos  nφ=cos  θ sin n φ=sin  θ

Rearranging: φ=θ +2kπn

Thus the kth root, ωk, for the complex number z for all k ≤ n − 1 (the maximum of n roots) is: ω k=ρ1/ n[cos (θ+2k πn)+ i sin( θ+2kπ n)]

This can be used to find solutions for roots for equations such as zn = m, where m is a complex number (and includes real numbers in the case of b = 0 in a + bi).
