Lecture 2: Elimination with Matrices

It is by the method of elimination that all the software packages solve linear equations. Consider: x+ 2y+z=2
3x+8y +z=12
&nbsp;&nbsp;&nbsp; &nbsp;4y+z=2

And the matrix:

[ 121 381 041 ]

Being of the A�" x= b type, this indicates that vector b is ( 2122 )

Everything is in the matrix. Now what does elimination do?

The algorithm for forward elimination is as follows. The pivot is the variable to be eliminated. Its coefficient is ar,c, where r is the pivot row and c is the pivot column.

The pivot factor is determined by ar+1,c + pivot factor &times; ar,c = 0, where ar+1,c is the coefficient in the next row, one column down. Doing the algebra, pivot factor = &minus;(ar+1,c &div; ar,c). In the current matrix, this means for the first elimination, the pivot factor = &minus;(3 &div; 1) = &minus;3

The next step is to compute all coefficients for row r + 1. This is simply: ar+1,c = ar+1,c + ar,c &times; pivot factor. Note that this equation says, take current coefficient of row r + 1 and add to it the product of the pivot factor and the row r coefficient corresponding to the variable (column).

Thus for row 2 of the matrix, the coefficients become:
3 + (&minus;3 &times; 1) = 0
8 + (&minus;3 &times; 2) = 2
1 + (&minus;3 &times; 1) = &minus;2

The matrix is now obtained

[ 121 02- 204 1]

The first pivot was at row 1, column 1. The next pivot will be at row 2, column 2. The pivot coefficient is 2, and the next row coefficient of the variable is 4. Thus the pivot factor = &minus;(4 &div; 2) = &minus;2.

Compute the coefficients of the next row:
4 + (&minus;2 &times; 2) = 0
1 + (&minus;2 &times; &minus;2) = 5

The elimination result is

[ 121 02- 200 5]

The result matrix is an upper triangular matrix, U. It should be clear that in calculating the pivot factor, the coefficient ar,c cannot be zero (it creates a divide by zero). It can also be noted that the determinant of the matrix U will be the product of all its pivot values: thus 1 &times; 2 &times; 5 = 10

When Does Elimination Fail

In the system of three linear equations above, failure of elimination occurs when three pivots cannot be obtained.

If one variable has coefficients of zero in any two equations (that is, the variable is not present), the it is vital that this equation be set as the first equation in creating the triangular matrix. Equations must be ordered so that non-zero pivots are obtained.

In the system above, what coefficient at row 2, column 2, or (2,2), would have caused failure? Changing from 8 to 6 would have resulted in a failure. A row exchange would have been required.

What about the pivot at (3,3), that is row 3, column 3? Had the value been &minus;4, that would have produced a zero-valued coefficient and thus failure.
