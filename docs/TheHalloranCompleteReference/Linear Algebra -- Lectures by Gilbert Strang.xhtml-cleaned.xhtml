Equations must be ordered so that non-zero pivots are obtained.</p> <p> In the system above, what coefficient at row 2, column 2, or (2,2), would have caused failure? Changing from 8 to 6 would have resulted in a failure. A row exchange would have been required.</p> <p> What about the pivot at (3,3), that is row 3, column 3?
