Transformation of expressions inside integrals is usually necessary.

The most common transformation is substitution .

But note the following:

  1. Do not neglect to substitute the dx .
  2. The function that is substituted must be one-to-one and continuous . Restrict the integral in a way, if necessary, to observe these requirements.
  3. When evaluating definite integrals, limits should be expressed in terms of the new dependent variable. With indefinite integrals, the reverse of the substitution must be done to obtain the answer in terms of the original independent variable.
∫ x 4 a 2 - x 2   d x

The substitution is x = | a | sin θ . Then d x = | a |   cos θ   d θ .

a 2 - x 2 = a 2 - a 2 sin 2 θ = | a |   1 - sin 2 θ = | a   cos θ |

The absolute value indicators should be noted: x 2 = | x | . This is always a common error in processing integration.

Note that the substitution in the example was not a one-to-one function, meaning that there is no unique inverse. The range of θ must be restricted to make the function one-to-one. This can be done easily by solving for θ :

θ = sin - 1 x | a |

Evaluation shows that the domain of the inverse is restricted to −π/2 to +π/2.

The integral becomes:

∫ a 4   sin 4 θ   | a |   cos θ | a | | cos θ | d θ

For the range of values for θ , cos θ is now always positive and so the absolute value signs can be removed from cos θ . The cos θ terms would cancel, and the resulting integral becomes:

a 4 ∫ sin 4 θ   d θ

Using integration tables, this integral evaluates as follows:

- a 4 { 1 4 ( sin 3 θ   cos θ ) + 3 8 ( cos θ   sin θ + θ ) } + C

The inverse substitution is now performed to get the result in terms of x . That is:

θ = sin - 1 x | a | sin θ = x | a |

Then:

cos θ = ± 1 - sin 2 θ = ± 1 - x 2 a 2 = ± a 2 - x 2 | a |

The ± sign can be omitted since cos θ is always positive. Now the substitution is complete, the final answer is:

∫ x 4 a 2 - x 2 d x = - 1 4 x 3 a 2 - x 2 - 3 8 a 2 x a 2 - x 2 + 3 a 4 8 sin - 1 x | a | + C

Any rational function of x can be integrated if the denominator is factored into linear and irrducible quadratic factors. This function may then be broken into partial fractions, and the individual partial fractions integrated by use of appropriate formula from the tables.

Integration by Parts

The other main method of integration is integration by parts . The general formula is

∫ u   d v = u ∫ d v - ∫ v   d u = u v - ∫ v   d u ∫ u   d v d x d x = u v - ∫ v   d u d x d x

The choice of functions u and v are critical. This means that v = ∫ d v and ∫ v   d u should be easier to integrate than the original integral.

Consider:

∫ x   sin   x   d x

Two obvious choices are u = x , d v = sin   x   d x or u = sin   x , d v = x   d x . A preliminary mental calculation shows that ∫ v   d u in the second choice is more complicated than the original integral since it would contain x 2 , so the first choice is used:

u = x , d u = d x , d v = sin   x   d x , v = - cos   x ∫ x   sin   x   d x = ∫ u   d v = u v - ∫ v   d u = - x   cos   x + ∫ cos   x   d x = sin   x - x   cos   x

In more complicated integrations, integration by parts may be combined with substitution or both may be used recursively to arrive at the answer.

Consider:

∫ e x   sin x   d x u = e x , then d u = e x d x ; d v = sin   x   d x , so v = - cos   x ∫ e x   sin   x   d x = ∫ u   d v = u v - ∫ v   d u = - e x   cos   x + ∫ e x   cos   x   d x

For the latter integral, repeat the integration by parts again:

u = e x , d u = e x   d x , d v = cos   x   d x , v = sin   x ∫ e x   sin x   d x = ∫ u   d v = u v - ∫ v   d u = - e x cos x + ∫ e x   cos x   d x         = - e x   cos x + ∫ u   d v         = - e x   cos   x + u v - ∫ v   d u         = - e x   cos   x + e x   sin   x - ∫ e x   sin   x

This would appear to be going back in a circular transformation. But it is not the case, since simple algebra will solve it:

∫ e x   sin   x = - e x   cos   x + e x   sin   x - ∫ e x   sin   x 2 ∫ e x   sin   x = - e x   cos   x + e x   sin   x ∫ e x   sin   x = 1 2 ( - e x   cos   x + e x   sin   x )

If the second integration by parts had been chosen such that u = cos   x , d v = e x   d x , then the circular transformation would have had no algebraic escape at the end. As a general rule never choose a substiution u at any stage to be the same as function v in a previous stage, or a constant time the previous function v .

An extended rule for integration by parts is, after the result of n + 1 successive applications of integration by parts, if:

g 1 ( x ) = ∫ g ( x )   d x , g 2 ( x ) = ∫ g 1 ( x )   d x , g 3 ( x ) = ∫ g 2 ( x )   d x , ... , g m ( x ) = ∫ g m - 1 ( x )   d x

then

∫ f ( x ) g ( x ) d x = f ( x ) g 1 ( x ) - f ' ( x ) g 2 ( x ) + f ' ' ( x ) g 3 ( x ) ... ... + ( - 1 ) n f n ( x )   g n + 1 ( x ) + ( - 1 ) n + 1 ∫ f n + 1 ( x )   g n + 1 ( x )   d x

Supposing that f ( x ) is a polynomial of degree n . Then f n + 1 ( x ) = 0 , and

∫ f ( x )   g ( x )   d x = f ( x )   g 1 ( x ) - f ' ( x )   g 2 ( x ) + f ' ' ( x )   g 3 ( x ) - ... + ( - 1 ) n f n ( x )   g n + 1 ( x ) + C

Suppose f ( x ) = x 2 and g ( x ) = sin x :

∫ x 2   sin   x   d x = - x 2 cos   x + 2 x   sin   x + 2 cos   x + C

Note the following points:

  1. A constant of integration is to be supplied with answers for indefinite integrals.
  2. Logarithmic expressions are to be base e (= 2.71828...) and to be evaluated for the absolute value of the arguments involved therein.
  3. All angles are measured in radians, and inverse trigonoemtric and hyperbolic functions represent principal values.
  4. If application of a formula produces a zero denominator or square root of a negative number, there is another form of the answer that usually avoids this difficulty.
