Type I, Type II Metal Cations

Type I Ions

Type I ions are metal elements that will only ever have ONE charge state. That state can be 1+, 2+, 3+ or whatever.

These metal elements happen to be

  • Group 1 elements Li+, Na+, K+, Rb+, Cs+, and Fr+
  • Group 2 elements Be2+, Mg2+, Ca2+, Sr2+, Ba2+, and Ra2+
  • Group 11 element Ag+, with 1+ charge
  • Group 12 elements Zn2+ and Cd2+
  • Group 13 elements Al3+ and Ga3+

Other Type I cations include Sc3+, Y3+, Lu3+

Type II Ions

Type II ions are metal elements that can have TWO or MORE charge states and include many typically transition metal elements. For example, cobalt (Co) can have a 2+ (Co2+) and 3+ (Co3+) state.

Because these ions can have more than one charge state, their names as used in compounds have a roman numeral corresponding to the charge state they have.

Co2+ forming compounds like CoCl2 will have the name cobalt(II) chloride, whereas Co3+ forming compounds like CoCl3 will have the name cobalt(III) chloride.

Only Type II cations have this roman numeral naming feature. Note that the parentheses containing the roman numeral immediately come after the element name, and there is no intervening space

Thus cobalt(II) chloride is correct and cobalt (II) chloride is not correct.
