LKB Rackbeta 1211 Liquid Scintillation Counter Operation

Description of Machine

The LKB-Wallac 1211 Rackbeta (made in 1985) is an automatic four-channel liquid scintillation counter that can count beta emissions in the energy range from 1 to 2800 keV. The 1211 can count at once 300 vials (thirty 10-vial racks).

It has a 15-key control panel:

  • Ten keys for the digits 0, 1, ..., 9. Before pressing a number, wait a short time to allow a previous digit key to return to its position.
  • A dot key to separate numbers
  • An R key for entering instructions or numbers into machine
  • A START key: press a group number, then R, then START to initiate automatic counting
  • A STOP key to stop any operation
  • An EDIT key to intiate a parameter setting mode
