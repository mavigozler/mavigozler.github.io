Numbers: Real, Rational, Irrational

The real numbers include all the numbers that elementary school students learn, an infinite decimal representation on an infinitely long number line. Real numbers include rational, irrational, algebraic, transcendental, positive, negative, and the number zero.

The set of real numbers is symbolized by ℜ ( ℝ )

The symbol R d refers to the domain of real numbers.

The symbol R n where n = 1, 2, 3, ... refers to n -dimensional space with real number coordinates.

Natural Numbers & Integers

A natural number is a number that represents the counting values: 1, 2, 3, ...

A natural number is a member of the set symbolized by ℕ

An integer is a value of the set of natural numbers that include zero and the negative set of natural numbers: (..., −3, −2, −1, 0, 1, 2, 3, ...)

An integer is member of a set symbolized by ℤ .

Rational and Irrational Numbers

A rational number is a number expressed as a quotient of a/b where a and b are integers, and denominator b cannot be equal to zero.

Integers themselves are also rational numbers since b can be equal to 1.

Rational numbers are represented by a decimal expansion that can be terminating or repeating. It is commonly believed that repetition without termination makes the number irrational, which is not the case.

An irrational number is basically a real number that does not conform to the criteria of rational number: it is a real number that cannot be expressed as a/b where b ≠ 0. The best known irrational numbers are π, e and √2. Decimal expansion of irrational numbers has no termination and does not repeat.

A rational number belongs to the set of numbers symbolized by ℚ

Cross-correlation

Cross correlation tests similarities of waveforms in which one is expressed as a time difference (or lag) of another. The test is also called the sliding dot product or innner product . The cross correlation of continuous functions f and g is defined as:

( f ⋅ g ) ( t ) = def ∫ - ∞ ∞ f ⋆ ( τ ) g ( t + τ ) d τ

f * is the complex conjugate of f .

The evaluation for discrete funcitons is

( f ⋅ g ) [ n ] = def ∑ m = - ∞ ∞ f ⋆ [ m ] g [ n + m ]

Cross correlation in signal processing is equivalent to shifting the signal and then multiplying it by another signal. It is somewhat similar to convolution , except that there is no first step involving the reversal of the signal.

Autocorrelation is cross correlation of the signal with itself.

Convolution and Deconvolution

Convolution is when a continuous function is operated by another continuous function to produce a third function.

The convolution of functions f and g is expressed as f * g . It is the integral of the product of two functions after one of them is reversed and then shifted:

( f ⋅ g ) ( t ) = def ∫ - ∞ ∞ f ( τ ) g ( t - τ ) d τ

It is also commutative:

( f ⋅ g ) ( t ) = def ∫ - ∞ ∞ f ( t - τ ) g ( τ ) d τ

The value t need not represent a time domain. But the convolution formula can be a weighted average of the function f ( τ ) at the moment t where weighting is given by g ( −τ ) simply shifted by the magnitude of t .

Fast Fourier Transform(ation)

The fast Fourier transform (FFT) is an algorithm to computer a discrete Fourier transform (DFT) and its inverse. There are several techniques that fall into this class of algorithm. The DFT isolates (decomposes) a sequence of values into components of different frequencies. computing a DFT of N points in the standard way requires O( N 2 ) operations; using the FFT, it takes O( N log N ) operations.

Let x 0 , ..., x N-1 be in the set of complex numbers. The DFT is defined as:

X k = ∑ n = 0 N - 1 x n   exp ( - i 2 π k n N )         k = 0 , ... , N - 1

This requires O ( N 2 ) operations to evaluate, that is, there are N outputs X k and each output requires a sum of N terms. An FFT requires O ( N log N ) operations to compute the same results. That is, a direct DFT calculation for N =10 points will require 10 2 = 100 operations, while an FFT will require 10 log 10 = 10 operations.

Cooley-Tukey Algorithm . This breaks down the a DFT recursively which has a composite size of N = N 1 N 2 into smaller DFTs of sizes N 1 and N 2 with O( N ) multiplicative operations using complex roots of unity called twiddle factors. The transform is divided into two pieces of size N /2 at each step

Metric spaces . A ( distance ) metric d( v 1 , v 2 ) ⇒ ℜ is defined such that

  • d is nonnegative: ∀ v i , v j ∈ V : d( v i , v j ) ≥ 0
  • d is symmetric: ∀ v i , v j ∈ V : d( v i , v j ) = d( v j , v i )
  • indiscernibles are identical: ∀ v i , v j ∈ V : d( v i , v j ) = 0 ⇔ v i = v j
  • the triangle inequality holds: ∀ v i , v j , v k ∈ V : d( v i , v j ) + d( v j , v k ) ≥ d( v i , v k )

Normed spaces . The length or norm of a vector is represented as ∥ v ∥

  • nonnegative: ∀ v ∈ V : ∥ v ∥ ≥ 0
  • positive definite: ∥ v ∥ = 0 ⇔ v = 0
  • symmetric: ∀ v ∈ V : a ∈ F : ∥ a v ∥ = | a | ∥ v ∥
  • the triangle inequality holds: ∀ v i , v j ∈ V : ∥ v i ∥ + ∥ v j ∥ ≥ ∥ v i + v j ∥

Banach spaces are normed spaces thate are complete, with no holes or missing points.

  • Real numbers, but not rational numbers, form a Banach space.
  • Euclidean n -space is Banach
  • p norm: ( ∑ i = 1 D | x i | p ) 1 p
  • p = 1: manhattan norm
  • p = 2: euclidean norm
  • Metric from norm: d( v 1 , v 2 ) = ∥ v 1 − v 2 ∥
  • The norm from metric if d is homogeneous: ∀ v i , v j ∈ V , a ∈ F : d ( a v i , a v j ) = | a | d ( v i , v j )
  • The norm from metric if d is translation invariant: ∀ v i , v j , t ∈ V : d ( v i , v j ) = d ( v i + t , v j + t )

Inner Product Spaces . The inner/scalar/dot product 〈 v i , v j 〉 ⇒ ℝ is defined such that

〈 v i + v j , v k 〉 = 〈 v i , v k 〉 + 〈 v j , v k 〉 〈 a v i , v j 〉 = a 〈 v i , v j 〉 〈 v i , v j 〉 = 〈 v j , v i 〉 〈 v , v 〉 ≥ 0 〈 v , v 〉 = 0 ⇔ v = 0
  • For complex spaces: 〈 v i , v j 〉 = 〈 v j , v i 〉 ¯ 〈 v i , a v j 〉 = a ¯ 〈 v i , v j 〉
  • Induces a norm: | | v | | = 〈 v , v 〉

For some inner products

  • Multiplication in ℝ
  • Dot Product in Euclidean n -space: 〈 v 1 , v 2 〉 = ∑ i = 1 D v 1 , i v 2 , i
  • For real functions over domain [ a, b ]: 〈 f , g 〉 = ∫ a b f ( x ) g ( x ) d x
  • For complex function over domain [ a, b ]: 〈 f , g 〉 = ∫ a b f ( x ) g ( x ) ¯ d x
  • Can add nonnegative weight function: 〈 f , g 〉 w = ∫ a b f ( x ) g ( x ) w ( x ) d x

Hilbert Space . An inner product space that is complete with respect to the induced norm is called a Hilbert space. This is an infinite Euclidean space. The inner product represents distances and angles. This is a subset of Banach spaces.

Orthogonality . Two vectors, v 1 and v 2 , are orthogonal if their inner product evaluates to zero: 〈 v 1 , v 2 〉 = 0

Vectors v 1 and v 2 are orthonormal if they are orthogonal and 〈 v 1 , v 1 〉 = 〈 v 2 , v 2 〉 = 1

For a set of vectors that are orthonormal, their inner products will evaluate to a constant called the Kronecker delta: 〈 v i , v j 〉 = δ i , j

The cosine series is defined as: f ( θ ) = ∑ i = 0 ∞ a i C i ( θ )

Evaluating for i = 0, 1, and n , and with a i = 1 to keep it simple:

C 0 ( θ ) = 1 C 1 ( θ ) = cos ( θ ) C n ( θ ) = cos ( n θ ) 〈 C m , C n 〉 = ∫ 0 2 π cos ( m θ ) cos ( n θ ) d θ = ∫ 0 2 π 1 2 ( cos [ ( m + n ) θ ] + cos [ ( m - n ) θ ] ) d θ

Note that for m ≠ n ≠ 0, then the evaluation will be:

( 1 2 ( m + n ) sin [ ( m + n ) θ ] + 1 2 ( m - n ) sin [ ( m - n ) θ ] ) | 0 2 π = 0

For m = n ≠ 0, then the evaluation will be:

∫ 0 2 π ( 1 2 cos ( 2 n θ ) + 1 2 ) d θ = ( 1 4 n sin ( 2 n θ ) + θ 2 ) | 0 2 π = π

For m = n = 0, then the evaluation will be:

∫ 0 2 π 1 2 2 cos ( 0 ) d θ = 2 π

Sine Series

The cosine series is defined as: f ( θ ) = ∑ i = 0 ∞ b i S i ( θ )

Evaluating this too for i = 0, 1, and n , and with b i = 1 to keep it simple:

S 0 ( θ ) = 0 S 1 ( θ ) = sin ( θ ) S n ( θ ) = sin ( n θ ) 〈 S m , S n 〉 = ∫ 0 2 π sin ( m θ ) sin ( n θ ) d θ

Now for the two cases of m ≠ n and m = n = 0, then: 〈 S m , S n 〉 = 0

For the case of m = n ≠ 0, then: 〈 S m , S n 〉 = π
