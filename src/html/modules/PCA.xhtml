Principal Component Analysis (PCA)

Prerequisite Knowledge

A knowledge of parametric statistics will be assumed, such as the fundamental calculation of mean, variance, and standard deviation of normally distributed data. Tests for normality exist to indicate if data is skewed or not.

Covariance and Covariance Matrix

Whereas the variance of a one-dimensional dataset is the sum of square of differences between data about its mean, the covariance is the sum of the product of the difference between two datasets and their mean:
var(x) = { ∑ (xi − x)(xi − x) } / (n − 1)

cov(x, y) = { ∑ (xi − x)(yi − y) } / (n − 1)

for all i from 1 to n. Note that if the computed value is <0, the increase in one dataset involves a decrease in values of the second dataset. If the covariance is 0, then the data are independent of one another.

In a multidimensional dataset, a pairing of each dimension can help in the computation of a covariance. For 3 dimensions, 3 covariance values can be computed. In fact for n dimensions, it is possible to compute n! / (2(n−2)!) covariance values. For all values of all dimensions, a covariance matrix can be constructed with n × n dimensions:

Cn×n = (ci,j, ci,j = cov(Dimi, Dimj).

A three-dimensional x, y, z covariance matrix would look:

C = ( cov(x,x) cov(x,y) cov(x,z) )
cov(y,x) cov(y,y) cov(y,z)
cov(z,x) cov(z,y) cov(z,z)
Note that in the matrix, the diagonal is just the variance of one dimension of the dataset. Also the matrix is symmetrical, since cov(a,b) = cov(b,a)

Matrix Algebra

Two matrices can be multiplied together if they meet certain requirements. Suppose a vector is multiplied to a square matrix, which is a transformation matrix. The result is another vector that is transformed from its original position. An eigenvector is a vector that is a multiple of a vector affected by a transformation matrix. Eigenvectors are properties of only certain square matrices: for such n × n matrices, there are n eigenvectors. Eigenvectors as such scale the magnitude of vectors they transform, without altering their direction. An additional property is that all eigenvectors are orthogonal (perpendicular) to the plane of the square matrix. Additionally eigenvectors are scaled to 1 so that they all have the same length.

PCA Defined

[Taken from one single page explanation by NGA Data Mining.]
  • Reduces dimensionality of large data sets, particularly useful in microarray analysis
  • Finds signals in noisy data
Suppose X is a matrix of data n rows (observations) × p columns (variables), x1, x2, ...,xp. There is another matrix S which is a p × p covariance matrix with the same set of xi variables.

A vector a1 of length p is found such that a1TSa1 is maximized subject to a1Ta1 = 1.

A variable z1 = ∑pi=1 a1i xi represents the first principal component, and provides a linear combination of variables yielding maximum variation. A second principal component, z2 = ∑pi=1 a2i xi is found with the condition that a2TSa2 is maximized subject to a2Ta2 = 0.

A linear combination of variables will now be orthogonal to the first principal component to provide maximum variation. Additional principal components are derived in a similar way.

Vectors a1, a2, ..., ap are eigenvectors of matrix S. Associated with each is an eigenvalue λi2. The value λi2 / ∑ λi2 is the proportion of variation explained by the ith principal component. Alternative, the ai values can be considered to be right singular vectors in a singular value decomposition with singular values λi of the matrix centered about its mean and scaled by 1 / √(n − 1), Xs.

If Xs = V Λ P´, with Λ being a diagonal matrix with elements λi, P´ is a p × p matrix with columns ai, and V is an n × p matrix with V´V = I, this produces the principal component scores.

In reducing the dimensions of data sets, one replaces a large number of correlated variables with smaller number of orthogonal variables containing most of the information of the data set. The choice of the count of dimensions is based on the variation of the leading principal components. If k principal components are selected, the test of equality for the remaining p − k eigenvalues is:

(n − 1 − (2p + 5) / 6 { − ∑p i=k+1 log(λi2) + (p − k) log ( ∑p i=k+1 λi2/(p − k) }

This has asymptotically a χ2 distribution with 1/2(p − k − 1)(p − k + 2) degrees of freedom. Equality in the remaining eigenvalues indicates that if more principal components are to be considered, they should all be considered.

Instead of the variance-covariance matrix, the correlation matrix, sum of squares and cross-products matrix, or a standardized sum of squares and cross-products matrix can be used. In the latter case σ−1/2Sσ−1/2 is used instead of S for a diagonal matrix σ with positive elements. If correlation matrix is used, the χ2 approximation for the statistic given above is not valid.

Principal component scores F are the values of the principal component variables for the observations. They can be adjusted (standardized) so that the variance of the scores for each component is 1.0 or equal to the corresponding eigenvalue.

In weighting components, the matrix X is first centered about the weighted means, then each row is scaled by an amount √wi where wi is the weight of the ith observation.
