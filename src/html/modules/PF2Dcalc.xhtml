Instructions

This is a special calculation program for the Beckman-Coulter PF2D system: it is for determining the fraction in the Gilson FC204 fraction collector containing the protein peak in which you are interested. Several parameters need to be entered to make that determination.

  • Volume or Time Per Fraction. You can choose to enter the value of the time (in minutes per fraction) or volume (in milliliters per fraction) that you are collecting in the FC204. This will usually be the setting on the FC204 that displays that value. Note that if you select a volume per fraction, the flow rate textbox input will appear and you must enter a flow rate of the HPRP 2nd dimension.
  • Start Collection Time. A required value is the time in minutes of the 2nd dimension program in which you start the collection of the 2nd dimension chromatography. Be very exact: if the time is 5.25 minutes, do not round it off to 5.3 minutes!
  • End Collection Time. As well, it is necessary to enter the time in minutes at which collection ends, even if it is the time at which the program ends. Again, exact values required.
  • Count of Tubes or Well Collected Per Cycle. This is a number that is programmed into the FC204. This indicates the maximum number of tubes or wells to be collected per cycle, and overrides any stop collection signal from an external controlling device. It is absolutely essential to enter this value if it is used.
  • Delay Between Detector and Collector. You can choose to enter a time, or perhaps better, to enter the dimensions of the tubing between the detector and the end of the tubing on the FC204 where it drops into the well. A reasonable default will be assumed if no entry is made here, but you are absolutely advised to enter the correct delay parameters, since no installation is standard.
  • HPCF fraction. You can enter, either in Row-Column (e.g., A1, A7, A10, B6, etc.) or Number of Fraction Collected format (1, 7, 10, 18, etc.), the fraction in which your protein was collected in the first dimension (chromatofocusing).
  • HPRP peak information. You can one of two kinds of information. One is the peak retention time: this is the actual top of the peak. Most users may do this. The other way is to enter the start time of when the peak begins its rise, and end time when you think it completes its detection, and perhaps when the peak width is long in time and may span more than one fraction. Spanning more than one fraction is quite possible if you measure the peak width at the half height in seconds or minutes, and your time per fraction is less than that.
After entering the required values, just click the Find Fraction button and the answer should be given. Alternatively, an error message might be displayed explaining why a determination could not be made.

In addition, the user can display a Thorn well map of the fraction collector multi-well plates, and view the fraction collected as a red-colored dot on the plate.

This module makes many assumptions. The two most important are:

  1. The multiwell plates are positioned in exactly the order shown, noting the A1 well position of each plate.
  2. The movement of the FC204 fraction collector arm is serpentine such that collection starts with A1 and proceeds in the following order:
    • A1 to A12, then to the next row to B12
    • B12 to B1, then to the next row to C1
    • C1 to C12, then to the next row to D12
    • continuining in this pattern shown
A future version of this module may be more programmable in that it lets the user define the motion/movement of the collector.
The Gilson FC204 Fraction Collector that collects the 2nd dimension eluate from the HPRP column into fractions is set to collect both a time per fraction and also to collect a maximum number of tubes/vials/wells per cycle. By setting the tubes/vials/wells per cycle, there is an assurance that partial fractions will not be collected.

To set or to view the number of tubes/vials/wells collected in a cycle, press the EDIT.
