How Much Of This Salt Do I Weigh Out?

This web page contains interactive features that basically permit users to determine what they need to do to make solutions for chemistry. This calculator can provide a description (short and long) about how much to weigh out of a salt and to prepare the solution. The inputs /include the concentration and volume the desired solution. There is no need to remember molecular weights: the user can provide a molecular formula or select from a list of commonly used chemicals, and the software embedded in the web page does the rest.

No Internet Connection Needed

There is no need to be connected to the Internet to use this page at all. All you need to do is to make sure that this web page is saved to the device where you maintain permanent storage. (For 99% of users, that is their hard disk.)

All of the content and software of this web page has been coded (most painstakingly, I might add) to run using client-side JavaScript. In order to use the interactive features of this web page, your client (browser) must therefore be enabled to execute scripts, specifically JavaScript. If you are reading this, then you are already enabled, so don't bother wondering how to do it.

Personalizing Your Copy

Is this cool or not!? I mean, the user can have this web page write his name at the top in big letters...impress his friends colleagues.

To do this, you need to have an editor that opens web pages for editing them. If you are using an operating environment other than a version of Microsoft Windows, you can ask someone how to find, open, and use an HTML editor. If you are using a version of Windows, do this:

  1. Start the simple text reader/editor called Notepad. If you know where it is on the Programs popup menu in the Start menu, then go there. A fast way to start it is to click Run... from the Start menu, then type in Notepad in the text entry space, then click `OK' or press ENTER.
  2. From within Notepad, use the File->Open dialog, look for the name of this file (HTML document). Notepad by default shows only files that have the type (extension) of .TXT as part of their filename. Either select the file type to show all files, or type an asterisk (* in the file name box to show all the files. Once you found the directory and file, open it by selecting and double-clicking it.
  3. Notepad will probably tell you that the file is too big, and it offer to open the file in Word Pad instead. Say Yes.
  4. In Word Pad (or whatever alternative Notepad offered), use the Find... command under the Edit menu to search for the word ‘Personalized.’ You should see a large coded comment that tells you what to do to personalize the page. Basically, you just put the name in quotes between an equal symbol and semicolon character. (If you want to know what it does, it assigns the string constant or the text in quotes to an identifier (variable) where it will be used later to write the text to the web page as it is filled.)

One other thing! Whatever HTML editor you use, make absolutely sure that it is not Microsoft Word! Yes, I know that Word edits and creates and saves as HTML documents, but the kind of coding that Word does to make HTML documents is positively awful! It is well known to web developers that Word creates the most unreadable and overly long web pages of any application that calls itself an HTML editor. Its so-called web pages can be rendered well enough (and that's a big guess) in Microsoft’s own browser, Internet Explorer, but other browsers have understandable trouble in rendering Word's positively dreadful HTML. You can open well-written pages in Word, and then immediately save them, and Word will save its own mangled HTML understanding of what the page looks like. Therefore, don’t use Word as an HTML editor....never!

This client-side web page was developed using two popular browsers (Microsoft Internet Explorer version 6 and Mozilla version 1.7). If something does not work when it is clicked, it is possible that the browser is one different than those tested, or that it is an older version of a browser that does not recognize code used to provide interactivity. The script makes use of functions and core objects in JavaScript 1.5 and level 1 of the Document Object Model as specified by the World Wide Web Consortium. In a few instances wrapper functions were written to provide workarounds with browsers that do not fully comply with the standards (although they are modestly called recommendations).

This page requires some external JavaScript and stylesheet files for support, which should have been installed if the install script/wizard was used. The files are placed within a path within the local filesystem; this path is //include for the script files, and //include/script for the external stylesheets. It is entirely possible to incorporate all script code and stylesheet descriptions within this one file, although the file size will surely be enormous. If this becomes a good idea (as indicated by several requests that it be done, or complaints that the page does not work), it is quite likely this page will be distributed as a single fully functional interactive web document.

There is the strongest interest in trying to make this page work on every system for every user that wants it to work. It is believed that the concepts taught and the tools provided are those used by and useful to many students and scholars of biochemistry and molecular biology. It is for this reason that where errors, faults, and exceptions occur, a routine is embedded to provide the user with the ease to send a message to the author of this system which contains information about the error or exception that has occurred. Use the Solutions Preparation interactive web page to get instructions for preparing a typical chemical solution, whether an acid, base, or salt. There are three necessary inputs:
  1. the molecular formula (to get the molecular weight of the chemical)
  2. the concentration (or solution density)
  3. the volume

For the molecular formula, there is the option of selecting from a list of common chemicals. If the chemical of interest is not in the list however, the molecular formula can be entered in a text box located next to an option called Other, with that option selected.

Molecular formulas can be complex. They must make use of recognized atomic symbols. Subscripts always apply to the atom subscripted, and radicals or groups of atoms can be grouped inside parentheses, with integers following the parentheses being subscripts for the grouping. Coefficients must be distinguished from subscripts with a leading comma and a trailing asterisk. Examples of atoms are given in the table to the right.
